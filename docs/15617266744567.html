如你所见，你正在访问的站点是 MWeb 静态网站主题 Reflect White 的示例站点。你可以在此尽情欣赏主题的细节设计。如果喜欢，不要忘记将它应用于你的 MWeb 生成的静态网站。

优秀的细节设计

整体主题使用大留白与杂志式排版风格，并以内容为中心，确保在带有良好设计感的同时兼顾阅读舒适性，读者不会被复杂设计吸引注意力。

移动设备也有良好表现

使用先进响应式设计，并针对小屏幕特别优化样式。即使是在移动设备上，也能保证良好的呈现效果。

完美兼容系统黑暗模式

主题全面兼容 Chrome 67+、Safari for macOS Mojave 以及 Safari for iOS 13 的亮暗适应，即使在黑夜也有良好的阅读体验。

良好的 Meta Tag 与 RSS Feed 支持

无需额外操作，主题已将 meta tag 和 RSS feed 文件安排妥当，读者无论是分享至社交网络、还是订阅网站，都能获得最佳效果。你需要考虑的，只有下一篇文章写什么。

多种个性化配置，调到满意为止

支持 <head> 标签自定义 JS、底部脚注、favicon 等多种个性化信息，原生支持 LikeCoin Widget 和社会化评论代码，你的网站由你做主。
